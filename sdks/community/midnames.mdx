# Midnames

## Compact Playground

You can access the playground at [playground.midnames.com](https://playground.midnames.com/). There, you can write and compile Compact code, and interact with it using the integrated Simulator.

You'll find the examples from the [Compact documentation](https://docs.midnight.com/compact) in the "Examples" section of the playground. You can also create your own projects and share them with others.

### Compilation

You can compile your Compact code either using a remote compiler (it sends the code to a server to be compiled) or a local compiler (it uses a WASM-based version of the compiler that runs in your browser). The latter has some limitations, such as failing to generate proper error messages.

:::caution[Local compilation]
Note that the local compiler is slow and may take a while to compile larger contracts, and the errors it generates are not helpful.
:::

## Nameservice

Midnames also runs a nameservice at [midnight.domains](https://midnight.domains/). The nameservice allows you to register human-readable names that resolve to Midnight addresses, making it easier to interact with contracts and addresses on the Midnight Network.

You can find a guide on how to buy a domain at [docs.midnight.domains](https://docs.midnight.domains/guides/buy_domain/).

### SDK

The midnight.domains SDK is available as an npm package. You can install it using:

```bash
npm install @midnames/sdk
```

The SDK provides functions to interact with the nameservice, such as registering a name, resolving a name to an address, and more. You can find the documentation for the SDK at [docs.midnight.domains](https://docs.midnight.domains/reference/resolve/).
