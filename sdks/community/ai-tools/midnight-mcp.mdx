# Midnight MCP

Midnight MCP is an open source MCP server purpose built for Midnight development. It gives AI coding assistants indexed knowledge of the Midnight ecosystem, real compiler validation, and semantic search across 102 repositories.

For a detailed walkthrough, see the [Midnight MCP blog post](/blog/midnight-mcp-ai-assisted-development).

## Quick Install

Add the following configuration to your AI assistant based on the tool you use.

### Claude Desktop

Edit `~/Library/Application Support/Claude/claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "midnight": {
      "command": "npx",
      "args": ["-y", "midnight-mcp@latest"]
    }
  }
}
```

### VS Code with GitHub Copilot

Edit `.vscode/mcp.json` in your project:

```json
{
  "servers": {
    "midnight": {
      "command": "npx",
      "args": ["-y", "midnight-mcp@latest"]
    }
  }
}
```

### Cursor

Edit `.cursor/mcp.json` in your project root:

```json
{
  "mcpServers": {
    "midnight": {
      "command": "npx",
      "args": ["-y", "midnight-mcp@latest"]
    }
  }
}
```

### Using with 1AM Wallet

The 1AM wallet extends Midnight MCP with wallet specific tools for transaction building, proof generation, and DApp integration. Install the 1AM wallet browser extension and configure your AI assistant to use it alongside Midnight MCP:

```json
{
  "mcpServers": {
    "midnight": {
      "command": "npx",
      "args": ["-y", "midnight-mcp@latest"]
    },
    "1am-wallet": {
      "command": "npx",
      "args": ["-y", "@1am/mcp-server"]
    }
  }
}
```

The 1AM wallet integration gives AI assistants access to wallet creation, balance queries, token transfers, contract deployment, and proof station interactions. For more details, see the [1AM documentation](https://docs.1am.xyz).

## Features

- **Real compiler validation**: Code is checked against the actual Compact compiler before you see it
- **Semantic search**: Query 102 repositories for patterns and examples
- **Version aware syntax**: Accurate Compact syntax for the current compiler version
- **29 tools**: Compilation, search, deployment, and testing utilities
