What is Midnight?

Midnight is a zero-knowledge partner chain to Cardano that revolutionizes blockchain privacy. Developed by Input Output Global (IOG), Midnight enables developers to build applications that can selectively hide sensitive information while still allowing public verification of correctness through advanced zero-knowledge cryptography.

Unlike traditional blockchains where all transaction data is publicly visible, Midnight allows smart contracts to operate on private data while maintaining the security and verifiability that makes blockchains valuable.

Core Value Proposition

Midnight solves the fundamental tension between transparency and privacy in blockchain technology, enabling entirely new categories of applications that were previously impossible.

Key Technologies

Compact Programming Language

A specialized programming language designed specifically for writing privacy-preserving smart contracts. Based on TypeScript syntax but compiled to zero-knowledge circuits.

  • TypeScript-based syntax
  • Statically typed with compile-time checking
  • Functional programming concepts
  • Compiles to ZK circuits and TypeScript definitions

Dual-Token System

NIGHT: Native utility token for governance, consensus, and block production rewards. Total supply: 24 billion tokens.

DUST: Shielded network resource for transaction fees. Decays over time to ensure privacy. Generated by staking NIGHT tokens.

Zero-Knowledge Proofs

Mathematical techniques that allow proving computations without revealing data. Currently transitioning from Pluto-Eris to BLS proving system.

  • ZK-SNARKs for privacy
  • Prevents transaction correlation
  • Enables selective disclosure
  • Regulatory compliance friendly

Partner Chain Architecture

Operates as a "partner chain" to Cardano, inheriting proven security while adding specialized privacy capabilities.

  • Inherits Cardano's security model
  • Cardano SPOs can become validators
  • Proven consensus mechanism
  • Dedicated privacy features

  • Containerization: Docker support for development services
  • Programming: TypeScript/JavaScript experience helpful
  • Resources: Sufficient to run a full non-mining node

Development Stack

  • Smart Contracts: Written in Compact language
  • DApps: Built using standard TypeScript
  • Proof Server: Generates zero-knowledge proofs
  • Midnight Node: Connects to the network
  • Lace Wallet: Browser extension for integration

Getting Started Process

  1. Set up development environment (Linux/macOS/Windows WSL)
  2. Install Compact compiler and development tools
  3. Connect to Midnight testnet
  4. Acquire test DUST (tDUST) tokens from faucet
  5. Follow tutorials to build your first application
  6. Deploy and test on testnet
  7. Iterate and improve your DApp

Running a Validator

Validator Overview

Midnight serves as a partner-chain to Cardano, presenting opportunities for Cardano Stake Pool Operators (SPOs) to help secure the Midnight network.

Requirements for Validators

  • Cardano SPO: Must operate a Cardano stake pool
  • Registration: Register as candidate in validator committee
  • Dual Operation: Run both Cardano and Midnight nodes
  • Re-registration: Required for major network upgrades
  • Rewards: Paid in NIGHT tokens
