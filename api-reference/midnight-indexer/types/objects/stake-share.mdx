Stake share information for an SPO.

Values are sourced from mainchain pool data (e.g., Blockfrost) and keyed by Cardano pool_id.

```graphql
type StakeShare {
  poolIdHex: String!
  name: String
  ticker: String
  homepageUrl: String
  logoUrl: String
  liveStake: String
  activeStake: String
  liveDelegators: Int
  liveSaturation: Float
  declaredPledge: String
  livePledge: String
  stakeShare: Float
}
```
