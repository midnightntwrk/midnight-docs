## Midnight 0.0.5 release notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features
- Made faucet send the drop with 5 outputs.  
- Added healthcheck for the faucet.  
- Covered faucet with extensive logs for better tracking.  

### Bug Fixes
- Removed missing links in the project.  

### Breaking Changes
_None_

### Known Issues
_None_

### Upgrade Instructions
- Ensure the faucet sends drops with 5 outputs as expected.  
- Verify that the healthcheck is functioning properly for the faucet.  
- Review the extensive logs for faucet activities to track any potential issues.  

### Other Changes
- Updated `README.md` for better clarity.  
- Bumped node image to the one allocating 5 outputs.  
- Updated wallet packages and some tooling.  
- Released version `0.4.2`.
