## Midnight 0.0.5 release notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features
- Created additional smoke test for Faucet.  
- Added medium-level automation tests for Faucet.  
- Updated Faucet to use Halo 2.  

### Bug Fixes
- Increased Lace timeout for improved reliability.  

### Breaking Changes
_None_

### Known Issues
_None_

### Upgrade Instructions
- Ensure that the new smoke and medium-level automation tests for Faucet are included in your test suite.  
- Verify that the Faucet is now correctly using Halo 2.  
- Test the Lace timeout changes to ensure stability under load.  

### Other Changes
- Updated Faucet version.  
- Updated test configurations for better compatibility and performance.
