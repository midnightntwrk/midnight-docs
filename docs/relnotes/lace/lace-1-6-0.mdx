# Midnight Lace wallet

Midnight Lace wallet is a specific Web3 light wallet platform developed for Midnight, tailored to managing assets and interacting with DApps on the Midnight blockchain, as well as giving users control over their personal data.

[Link to related documentation](https://www.lace.io/)

## Midnight Lace wallet 1.6.0 Release Notes

Learn about the latest updates and improvements to the Midnight Lace wallet, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- First Midnight Lace release, integrating with the following Midnight dependencies:
  - `@midnight/wallet:3.2.1`
  - `@midnight/wallet-api:3.1.3`
  - `@midnight/dapp-connector-api:3.1.2`
  - `@midnight/zswap:0.3.2`

### Known Issues

- Version 1.6.0 is set due to an issue with the manifest key, which will be addressed in future releases.

### Upgrade Instructions

1. Download and unpack the attached `.tgz` file.
2. Run Google Chrome in developer mode.
3. Load the extension from the unpacked folder.

### Other Changes

- Initial release as `1.6.0`.
