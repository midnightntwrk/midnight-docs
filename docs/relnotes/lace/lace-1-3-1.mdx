# Midnight Lace wallet

Midnight Lace wallet is a specific Web3 light wallet platform developed for Midnight, tailored to managing assets and interacting with DApps on the Midnight blockchain, as well as giving users control over their personal data.

[Link to related documentation](https://www.lace.io/)

## Midnight Lace wallet 1.3.1 Release Notes

Learn about the latest updates and improvements to the Midnight Lace wallet, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Implemented performance testing and proof of concept (POC) for performance testing to validate system scalability and efficiency.  
- Enabled WARN logging for the ready endpoint to improve visibility into the system's health and status.

### Bug Fixes

- Adjusted DB connection test to ensure accurate validation and handling of database connectivity.
- Handled errors during wallet saving to ensure smoother operation and prevent crashes.

### Upgrade Instructions

- Ensure that performance testing is integrated into your workflow for continuous scalability validation.
- Verify that the WARN logging level for the ready endpoint is providing sufficient information for monitoring.
- Check that DB connection tests and wallet saving error handling are functioning correctly to prevent failures.

### Other Changes

- Simplified the wallet-zswap dependency for a more efficient and manageable integration.
- Updated Scala dependencies for improved performance and compatibility.
