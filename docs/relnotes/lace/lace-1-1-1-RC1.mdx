# Midnight Lace wallet

Midnight Lace wallet is a specific Web3 light wallet platform developed for Midnight, tailored to managing assets and interacting with DApps on the Midnight blockchain, as well as giving users control over their personal data.

[Link to related documentation](https://www.lace.io/)

## Midnight Lace wallet 1.1.1 RC1 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added tests against Ruby.  
- Added native tokens tests.  

### Bug Fixes

- Fixed syncing test for undeployed environments.  
- Fixed native tokens tests.  
- Fixed Midnight workflows.  
- Fixed release workflow for Midnight.  
- Fixed nightly inputs.  
- Fixed `midnight wallet` password management.  

### Breaking Changes

- Removed `devnet` and `qanet` environments, replaced with `pino` logger.  

### Upgrade Instructions

- Ensure compatibility with the updated workflows and new testing approaches.  
- Test native token functionality after the fixes applied to native token tests.  
- Verify the changes to the release action and workflow processes.  

### Other Changes

- Released version with Midnight release action fixes.  
- Reorganized Midnight workflows for better structure.  
- Switched build action to use `yarn build`.  
- Added version variable for better control of versioning.
