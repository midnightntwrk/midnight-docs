# Midnight Lace wallet

Midnight Lace wallet is a specific Web3 light wallet platform developed for Midnight, tailored to managing assets and interacting with DApps on the Midnight blockchain, as well as giving users control over their personal data.

[Link to related documentation](https://www.lace.io/)

## Midnight Lace wallet 2.0.0 RC3 Release Notes

Learn about the latest updates and improvements to the Midnight Lace wallet, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added support for Halo2 viewing keys to improve the functionality and integration with the Halo2 environment.
- Implemented timeout checks to enhance the system's resilience and ensure better management of timeouts during critical operations.
- Added a check for the latest finalized block to improve synchronization and data integrity within the system.
- Configured the Halo2 QA environment to streamline testing and enhance consistency in the QA process.

### Bug Fixes

- Fixed flaky tests to improve stability and reliability of the test suite.
- Excluded tracing from coverage to prevent false negatives during coverage reports.

### Known Issues

- Some contract deployments have been temporarily skipped while we work on further updates.

### Upgrade Instructions

- Verify that the Halo2 viewing keys integration is working as expected in your environment.
- Ensure that timeout checks and the latest finalized block check are properly implemented to handle synchronization issues effectively.
- Confirm that the Halo2 QA environment configuration is up to date and ready for use in testing.

### Other Changes

- Updated Scala dependencies for improved performance and compatibility.
- Improved npm authentication for better management of dependencies in the project.
