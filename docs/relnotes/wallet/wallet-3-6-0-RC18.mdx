## Wallet 3.6.0 RC18 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

#### New Features

- Added `hardfork-qa` workflows and included native tokens in funding.  
- Added `halo2` wallet health check.  

### Bug Fixes

- Fixed seed used in the funding workflow.  

### Breaking Changes

- Refactored to raw `SyncService`.  
- Supported multiple ledger versions.  
- Updated to `zswap 3.0.0-beta.3`.  


### Upgrade Instructions

- Ensure compatibility with the updated `SyncService`.  
- Update integrations to support multiple ledger versions.  
- Verify the correct use of seeds in the funding workflow.  
- Test native token funding in `hardfork-qa` workflows.  

### Other Changes

- Released version `v3.6.0-rc.18`.
