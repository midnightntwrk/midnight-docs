## Wallet 3.5.6 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Scheduled health checks for Jade and Topaz environments.  


### Upgrade Instructions

- Update to wallet version `3.5.5`.  
- Ensure scheduled health checks are running for Jade and Topaz.  
- Remove dependencies or configurations related to Devnet and Qanet, as they have been deprecated.  
- Verify that Maven packages are now published to `midnight-ntwrk`.  

### Other Changes

- Improved end-to-end tests.  
- Fixed issues related to publishing to `midnight-ntwrk`.  
- Released version `3.5.6`.
