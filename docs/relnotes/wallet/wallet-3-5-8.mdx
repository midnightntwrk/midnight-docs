## Wallet 3.5.8 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

#### New Features

- Added tests against Ruby.  

### Bug Fixes

- Used sync viewing updates index to compare against sync progress total.  
- Validated viewing key on deserialization.  

### Breaking Changes

- Removed legacy wallet packages.  


### Upgrade Instructions

- Ensure compatibility with the updated sync viewing updates index.  
- Verify that viewing key validation on deserialization does not affect existing functionality.  
- Migrate away from any dependencies on legacy wallet packages, as they have been removed.  
- Update c-bindings and ensure compatibility with the new version.  

### Other Changes

- Updated c-bindings and bumped the version.
