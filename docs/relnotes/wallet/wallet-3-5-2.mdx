Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### Upgrade Instructions

- Ensure WebSocket connections are correctly handled for initialization and subscription.  
- Update dependencies to the latest versions and verify e2e tests run as expected.  
- Remove any `.only` flags from tests if present to ensure full test execution.
