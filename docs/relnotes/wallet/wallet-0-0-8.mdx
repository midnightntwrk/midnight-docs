## Wallet 0.0.8 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added publishing for `ogmios-sync` JAR.
- Configured publishing to include only `ogmios-sync`.

### Bug Fixes

- Adjusted the order of publishing artifacts.

### Upgrade Instructions

- Ensure the updated publishing configuration is in place for `ogmios-sync`.
- Verify Cicero config integration for publishing.
