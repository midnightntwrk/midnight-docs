## Wallet 2.0.0 release notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added Ledger integration. 
- Linked new `Wallet-API` and `Ledger-API`.
- Added option to create a wallet with an initial state.

### Upgrade Instructions

- Ensure compatibility with the new `Wallet-API` and `Ledger-API` integration.
