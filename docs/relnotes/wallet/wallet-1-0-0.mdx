## Wallet 1.0.0 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added events for the `ogmios-*` modules.
- Added events for wallet modules.

### Breaking Changes

- Updated the dependency to the new `wallet-api`.

### Upgrade Instructions

- Review the ADR regarding the lack of mutation tests (`Stryker`).
- Improved code coverage across the project.
