## Wallet 3.5.12 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added new C binding for filtering `zswap` state.  

### Bug Fixes

- Removed IOG Nix cache from CI.  

### Breaking Changes

- Removed Nix ledger package.  


### Upgrade Instructions

- Ensure compatibility with the removal of the Nix ledger package.  
- If using `zswap` state filtering, update to the new C binding implementation.
