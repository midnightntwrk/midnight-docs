## Wallet 3.6.0 RC15 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### Bug Fixes

- Increased proving timeout for RC2 version.  
- Removed built wallet dependency.  

### Upgrade Instructions

- Ensure compatibility with the updated proving timeout in RC2.  
- Verify that the removal of the built wallet dependency does not impact existing functionality.
