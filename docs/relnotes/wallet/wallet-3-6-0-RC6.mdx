## Wallet 3.6.0 RC6 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added a fresh test wallet for `halo2-qa`.  
- Added `bboard` wallet.  
- Implemented hard fork updates.  

### Bug Fixes

- Fixed branch for `halo2` health check.  

### Breaking Changes

- Removed Nix from CI/CD.  
- Upgraded Yarn to v4 and Node.js to v20.  

### Upgrade Instructions

- Ensure compatibility with the removal of Nix from CI/CD.  
- Update dependencies to support Yarn v4 and Node.js v20.  
- Verify `wallet-api` integration after the new package exports update.
