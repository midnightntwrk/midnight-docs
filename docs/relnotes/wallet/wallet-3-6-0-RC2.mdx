## Wallet 3.6.0 RC2 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### Bug Fixes

- Increased the timeout of the proof server client from 5 to 10 minutes.  


### Upgrade Instructions

- Ensure any integrations relying on the proof server client can handle the increased timeout.
