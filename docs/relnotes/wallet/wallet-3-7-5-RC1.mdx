## Wallet 3.7.5 RC1 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added support for Node/Rust indexer protocol version.  
- Bumped `zswap` to version `3.0.6`.  

### Bug Fixes

- Added missing `sbt` in GitHub Actions.  
- Added Checkmarx scan for security scanning.  


### Upgrade Instructions

- Ensure compatibility with the updated `zswap` version (`3.0.6`).  
- Integrate the new support for Node/Rust indexer protocol version where applicable.  
- Review and configure Checkmarx scan integration for security.  

### Other Changes

- Created `CODEOWNERS` for repository ownership.  
- Added an option to run Continuous Deployment (CD) from another branch.
