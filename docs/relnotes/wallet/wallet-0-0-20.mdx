## Wallet 0.0.20 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Removed `LaresService` (Racket server).

### Bug Fixes

- Changed `node-fetch` to `cross-fetch` in `stIgnore`.
- Removed integration tests from `flake.nix`.
- Updated tests in `midnight-wallet/integration-tests` to run on Node instead of Selenium and enabled integration test scope in modules using ScalaJS.

### Upgrade Instructions

- Ensure the new `ogmios-core` module is used for shared code.
