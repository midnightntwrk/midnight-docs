## Wallet 3.6.3 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.


### Bug Fixes

- Updated end-to-end tests for `3.0.0` ledger.  
- Bumped `zswap` dependency to `3.0.1`.  

### Breaking Changes

- Bumped dependencies to `ledger 3.0.2`.  


### Upgrade Instructions

- Ensure compatibility with the updated `ledger 3.0.2`.  
- Verify that the end-to-end tests are working correctly with the `3.0.0` ledger update.  
- Update to the latest `zswap` dependency (`3.0.1`).  

### Other Changes

- Released version `3.6.2`.  
- Applied minor improvements.
