## Wallet 3.5.10 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added native token support on `ariadne-temp` and expanded addresses in the funding script.  
- Introduced `waitForTxInHistory` helper for transaction testing.  
- Implemented a workflow to fund a specific wallet across all environments.  
- Added additional end-to-end tests.  

### Bug Fixes

- Fixed native token tests.  
- Fixed waiting for transactions in history in transfer tests.  
- Fixed `dapp-connector` TypeDoc build.  
- Fixed workflows to work correctly with Yarn.  
- Addressed step condition issues in workflows.  
- Set progress status to `undefined` when connection is lost.  

### Upgrade Instructions

- Ensure the updated sync condition now progresses only when total equals synced.  
- Use the new workflow for funding specific wallets across all environments.  
- Update to the latest `dapp-connector-api` documentation.  
- Verify updates to the `ariadne-qa` URLs.  
- Update and replace DAO wallet if applicable.  
- Ensure the updated Welcome wallet is configured correctly.
