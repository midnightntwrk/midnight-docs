## Wallet 3.6.0 RC17 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Stopped syncing gracefully when the wallet is closed.  

### Upgrade Instructions

- Ensure wallet syncing stops as expected when closing the wallet.  
- Verify Renovate configuration if using dependency updates.  
- Update the counter wallet if applicable.
