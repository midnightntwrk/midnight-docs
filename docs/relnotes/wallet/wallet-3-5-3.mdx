## Wallet 3.5.3 Release Notes

Learn about the latest updates and improvements to the Midnight node, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

### New Features

- Added workflows for devnet and qanet health checks.  
- Updated native token fee calculation to use `tdust`.  

### Bug Fixes

- Updated latest height only on new viewing updates.  

### Upgrade Instructions

- Ensure the new workflows for devnet and qanet health checks are configured if applicable.  
- Verify native token fee calculations now use `tdust`.  
- Regenerate and review updated documentation based on configuration tweaks.
