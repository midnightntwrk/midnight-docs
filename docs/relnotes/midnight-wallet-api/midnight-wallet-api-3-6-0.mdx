## Midnight Wallet API 3.6.0 Release Notes

Learn about the latest updates and improvements to the Midnight Wallet API, including new features, bug fixes, and breaking changes designed to enhance functionality and stability.

#### New Features  

- Exposed additional fields for Bech32 addresses.  
- Updated API documentation.  
- Updated ZSwap to version 3.0.3, later reverted to 3.0.2.  
- Updated **nixpkgs** to **nixos-24.11**.  
- Updated **renovate.json** for SRE instance.  

#### Bug Fixes  

- Fixed Docusaurus character compatibility issue.  
- Fixed public/private README inconsistencies.
