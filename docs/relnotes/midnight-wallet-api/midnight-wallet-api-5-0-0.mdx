# Midnight Wallet API

Midnight Wallet API provides a comprehensive interface for wallet operations, defining the structure of the wallet state, the methods for interacting with it, and the types and variables used within.

To learn more, see the [Wallet API documentation](../../develop/reference/midnight-api/wallet-api/)

## Midnight Wallet API 5.0.0 Release Notes

### Stable Release

* **Version**: 5.0.0
* **Audience**: Wallet developers, integration engineers, node operators
* **Environment**: Production-ready


This release (version `5.0.0`) is considered stable and production-ready. It can safely be deployed in public and production environments.

### Key Changes and Highlights

* BLS Support
* Adds new sync progress API
