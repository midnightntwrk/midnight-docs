As a smart contract developer, you are responsible for building privacy-preserving smart contracts using Compact.
You'll use the Midnight toolchain to create and test your smart contracts, and then deploy them to the Midnight Network.

## Prerequisites

- Node.js (v20 or higher)
- Command-line knowledge
- Code editor (e.g. Visual Studio Code)
