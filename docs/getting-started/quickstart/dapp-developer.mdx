# DApp developer

As a DApp developer, you are responsible for building and deploying privacy-preserving DApps on Midnight. 
You'll use the Midnight toolchain to create and test your DApps, and then deploy them to the Midnight network.

## Prerequisites

- Node.js (v20 or higher)
- Command-line knowledge
- Code editor (e.g. Visual Studio Code)
